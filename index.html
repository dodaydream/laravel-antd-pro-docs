Laravel Antd Pro

基於 Laravel, Ant Design Vue 和 Inertia.js 的後臺快速開發框架

快速開始 介紹

快速構建 CRUD

簡單幾行代碼，便可構建功能完整的 CRUD 介面

背景任務管理器

讓耗時的任務於後臺完成，無需打斷正在進行的工作流程，並可隨時跟蹤進度

系統通知

內建系統通知功能，將消息更新即時傳達到用戶

# 快速開始

git clone https://github.com:dodaydream/laravel-antd-pro

cd laravel-antd-pro
cp .env.example .env

yarn install
composer install

php artisan serve
yarn watch
